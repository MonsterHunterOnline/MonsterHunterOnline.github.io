Installation guide via TGP
This guide assumes that you have already set up a QQ account.
  1. Go Here and download TGP (Click the big blue button).

  2. Go Here and download the latest Locale-Emulator.

  3. Extract the Locale-Emulator program with your favorite file extractor. (I use 7-zip because it's free.)
  4. Click "Install/Upgrade" if it warns you about japanese things ignore and click install on the next screen too.
  5. Run the LEGUI.exe program to make your locale chinese. Select the settings below and click "Save".

  6. Right click the tgpminiloader file you downloaded earlier and select "Run in Chinese".
  7. When it opens click the bottom right button.
  8. Put an install location in, or leave the current one if you so fancy, unselect the two check marks, and click the bottom right button here too. After that press the blue button in the middle.
  9. Wait for it to install...

  10. Click the big green button.

  11. Click next a bunch of times.




  12. Click Finish!
  13. Log in to your account and click the second option in the TGP window.
  14. Click the third tab down and then on MHO.
  15. Another damn green button.
  16. Wait for download to finish. This will be way faster than using the old method which we won't talk about.
  17. Once downloaded it will install in the background. You will hear the wonderful Monster Hunter theme music in the background while it does this...
  18. When the Youku bs comes up close out by the x at the top then go to your taskbar, right click it and click the bottom button.
  19. Uninstall that shit. (Unless you are into Chinese things. Like that.)
  20. At the same time you were uninstalling Youku this little popup probably happened. this means it was installed and you can run the game!

    21. If you installed it during maintenance you will see this:
  21. The game will update if it is not in maintenance. If another popup happens when you press the button, this is TGP asking if you would like to authorize TGP to keep Monster Hunter Online updated. Click the left button which will be Yes!
