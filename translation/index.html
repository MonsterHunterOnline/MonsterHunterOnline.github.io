☰ Translation

Reaver's Super Sexy Translation Solution! (Android only because he doesn't believe in iPhones)

What you need:

  1. Android phone
  2. Root access to said phone (Root access is only so Tasker can do screen touches for you)
  3. A Dropbox account
  4. These apps:
    1. Dropsync
    2. Translate
    3. Tasker
  5. This script

Setting up Dropbox

You will need to set up Dropbox to sync with a folder on your computer.

  1. Log in to your Dropbox account (create one if you haven't already).
  2. Click the drop down menu where your name is displayed (top right).
  3. Click "Install"
  4. Download and install the file.
  5. Sign in to your Dropbox account from your computer.

Now you will need a folder that your phone will grab the files from. So open the Dropbox folder in Windows explorer and create a folder (Mine is named "Translation")

This is where you will save images to be translated.

Setting up Dropsync

  1. Add your Dropbox account to Dropsync.
  2. In the SYNCED FOLDERS tab click the + icon to create a new synced folder.
  3. Select a folder on your device that you will use for storing images locally, and select your "Translation" folder in Dropbox.
  4. Either select "Download only" or "Download then delete" it all depends on if you want to keep the translation images in your Dropbox.
  5. Save the Folder Pairing.

Setting up Translate

  1. Open translate and switch the "translate from" to Chinese, and the "translate to" to English (or whatever language you like best)
  2. Click the Camera Icon.
  3. Click the Photos Icon on the bottom left.
  4. Navigate to your local "Translate folder"
  5. Exit Translate app

Setting up Tasker

  1. Open Tasker and go to the TASKS tab.
  2. Press TASKS tab again and import downloaded task xml.
  3. Click on the Translate task.
  4. Click the run button to run through the first time to give root access.
  5. Close out of Tasker using drop down menu and "Exit".

Making a shortcut

  1. Add a widget to home screen
  2. Select "Task Shortcut 1 x 1" in Tasker section
  3. Select the Translate task and back out.

Now when you select that Translate widget, it should sync your Dropbox folder, open Google Translate and go to the screen where you can select the images. If your phone is slower than mine, you may have to go into the task and adjust the time under the "Wait" task.

To translate some text, use "Snipping Tool" to grab the section of your screen and save it in your "Translation" folder. Then open the Translate widget we just created and select your image!
